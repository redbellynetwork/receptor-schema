# beneficiaryNames

The names of Beneficiaries linked to the business in a comma separated format.

```
  "beneficiaryNames": "John Doe,Jane Smith"
```

# beneficiaryRoles

The roles of Beneficiaries linked to the business in a &(ampersand) separated format and each beneficiary roles seperated by commma.

Like if there are 2 users and 1st one has ubo and director roles and 2nd one has ubo and representative roles

```
  "beneficiaryRoles": "ubo&director,ubo&representative"
```
