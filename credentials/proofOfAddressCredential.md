```
  "country": "IND"
```

# city

The city of residence provided in address proof.

```
  "city": "Chandigarh"
```

# street

The street name provided in address proof.

```
  "street": "Lane 1"
```

# postCode

Represents the post code issued in the document of address proof.

```
  "postCode": "120991"
```
